DATA

데이터(Data)란?

문자, 숫자, 기호 등의 형태로 된 의미 단위이다. 컴퓨터가 다루는 모든 정보는 결국 데이터로 표현된다.

UNIT

데이터 단위

데이터는 크기에 따라 아래와 같은 단위로 나뉜다.

단위정의크기
비트 (Bit)0 또는 1TRUE / FALSE
1 바이트 (Byte)8 Bits영문자/숫자 1글자 : 1 Byte
한글 1글자 : 2 Bytes
1 킬로바이트 (KB)1,024 Bytes장문의 텍스트
1 메가바이트 (MB)1,024 KB저용량 이미지, 음악 파일
1 기가바이트 (GB)1,024 MB20~40분 길이의 동영상 파일
1 테라바이트 (TB)1,024 GB약 200개의 동영상 파일

비트 8개가 모이면 1바이트가 된다. 그림으로 보면 이렇다.

0101 1001

0과 1이 8개

➜
1 BYTE
정리하면 👉 0과 1(비트) 8개가 모여야 비로소 문자 하나를 표현할 수 있는 1바이트가 된다.
TYPE

자료형(Data Type)이란?

데이터를 표현하는 방법, 즉 데이터의 종류이다. 자료형은 메모리 공간의 크기와 자료의 형태를 정의한다.

int
123
float
3.14
char
'A'
string
"안녕하세요"
bool
true/false

💠 C++의 자료형 종류

문자형
char, wchar_t
char16_t, char32_t
정수형
short, int
long, long long
실수형
float, double
long double
논리형
bool
참고 👉 C++에는 Byte 타입이 따로 없고, char가 보통 1바이트 크기로 사용된다.

자료형별 설명과 예제는 다음과 같다.

자료형설명예제
int정수 (소수점 없는 숫자)10, -3, 1000
float실수 (소수점 있는 숫자)3.14, -0.5
char문자 하나'A', 'B', 'C'
string여러 글자"Hello", "C++ 공부"
bool참/거짓true, false
VARIABLE

데이터와 변수의 관계

자료형만으로는 데이터를 "어떻게 저장할지"만 정해질 뿐, 그 데이터를 다시 꺼내 쓸 방법이 없다. 그래서 변수(Variable)라는 이름표를 붙여 데이터를 담아둔다. 

int age = 20;
  • int — 이 상자에 어떤 종류의 데이터가 들어갈지 정하는 자료형이다.
  • age — 상자에 붙이는 이름표, 즉 변수명이다.
  • 20 — 상자 안에 실제로 담기는 데이터(값)이다.
여백 메모 ✏️ 자료형은 "상자의 종류", 변수는 "이름이 붙은 상자", 데이터는 "상자 안에 든 내용물"이라고 정리하면 세 개념이 헷갈리지 않는다.
DETAIL

자료형별 크기와 범위

같은 정수형이라도 크기(바이트)에 따라 표현할 수 있는 범위가 다르다. 필요한 만큼만 적절한 자료형을 골라 쓰는 것이 메모리를 절약하는 방법이다.

자료형형태크기(byte)범위
bool논리1true(1) / false(0)
char정수(문자)1-128 ~ 127
wchar_t문자20 ~ 65,535
char16_t문자20 ~ 65,535 (UTF-16)
char32_t문자40 ~ 4,294,967,295 (UTF-32)
short정수2-32,768 ~ 32,767
int정수4-2,147,483,648 ~ 2,147,483,647
long정수4-2,147,483,648 ~ 2,147,483,647
long long정수8약 -9.2경 ~ 9.2경
float실수4소수점 6자리 정밀도
double실수8소수점 15자리 정밀도
long double실수8구현에 따라 다름
실전 팁 👉 정수는 보통 int, 소수는 double을 기본으로 쓰고, 메모리를 아껴야 할 때만 short/float처럼 더 작은 자료형을 고려한다.
