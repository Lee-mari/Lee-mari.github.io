변수(Variable)란?

데이터를 저장할 수 있는 메모리 공간이다.

  • 변수는 이름표가 붙은 상자라고 생각하면 된다.
  • 이 상자 안에 값을 넣고, 필요할 때 값을 꺼내서 사용할 수 있다.

상자를 눌러보면 안에 든 값이 뒤집히면서 나타난다. 👇

name
눌러보기
"마리"
catName
눌러보기
"코코"
age
눌러보기
3
(마우스를 올리면 상자가 뒤집히며 안의 값이 보인다)
DECLARE

1. 변수 선언

변수를 선언하면, 그 이름으로 데이터를 저장할 공간을 만드는 것이다.

#include <stdio.h>

int main()
{
    int age;   // 정수를 담을 상자를 만든다

    return 0;
}
int
정수를 담을 수 있는 타입
➜
age
상자의 이름
int age;
비어있는 정수 상자 (아직 값이 들어있지 않음)
1
타입 선택 (int)

어떤 종류의 데이터를 담을 상자를 만들지 결정한다. int는 정수를 담을 수 있는 상자를 의미한다.

2
변수명 지정 (age)

상자에 이름을 붙인다. 나중에 이 이름으로 상자를 찾을 수 있다.

3
선언 완료 (int age;)

세미콜론(;)으로 선언을 완료한다. 이제 정수를 담을 수 있는 'age'라는 이름의 빈 상자가 만들어졌다.

INITIALIZE

2. 초기화

변수에 지정한 값을 저장하는 것이다. 변수에 값을 넣는 것을 "값을 할당한다"고 한다.

1단계: 빈 상자 만들기 (선언)

age
(비어있음)
height
(비어있음)

2단계: 상자에 값 넣기 (초기화)

age
10
height
156.7
#include <stdio.h>

int main()
{
    int age;         // 나이라는 상자를 만든다
    float height;    // 키라는 상자를 만든다

    age = 10;        // 나이라는 상자에 10을 넣었다
    height = 156.7;  // 키라는 상자에 156.7을 넣었다

    return 0;
}
COMBO

3. 선언과 동시에 초기화

변수를 선언함과 동시에 초기화를 할 수 있다.

#include <stdio.h>

int main()
{
    int age = 10;
    return 0;
}
여백 메모 ✏️ "선언"은 상자를 만드는 것, "초기화"는 그 상자에 처음 값을 채우는 것이다. 두 동작을 한 줄로 합친 것이 "선언과 동시에 초기화"이다.
WHY

3-1. 선언과 동시에 초기화하면 좋은 점

1️⃣ 쓰레기 값이 들어가는 것을 방지할 수 있다

변수를 선언만 하고 초기화하지 않으면, 컴퓨터 메모리에 남아있는 이상한 값(쓰레기 값)이 들어갈 수도 있다.

#include <stdio.h>

int main()
{
    int age;  // 초기화 안 함
    printf("나이: %d\n", age);  // 쓰레기 값이 출력될 가능성 있음!
    return 0;
}

2️⃣ 코드가 더 예측 가능해지고, 오류를 줄일 수 있다

초기화하지 않은 변수를 사용하면, 값이 정해져 있지 않아서 프로그램이 엉뚱한 동작을 할 수 있다.

#include <stdio.h>

int main()
{
    int num;      // 초기화 안 함!
    num += 5;     // 원래 값이 뭔지 모른 상태에서 5를 더함
    printf("%d\n", num);  // 예상하지 못한 결과가 나올 수 있음!
    return 0;
}

3️⃣ 코드가 간결하고 가독성이 좋아진다

한 줄에 선언과 초기화를 같이 하면, 코드를 더 짧고 읽기 쉽게 만들 수 있다.

📌 비효율적인 코드 
int a;
a = 10;
📌 좋은 코드 
int a = 10;
USE

4. 변수의 값 사용하기

변수에 저장된 값을 꺼내서 사용할 수 있다.

#include <stdio.h>

int main()
{
    int age = 10;
    float height = 179.8;

    printf("내 나이는 %d 살이고, 키는 %.1f 이다.\n", age, height);
    return 0;
}

== 결과 ==

내 나이는 10 살이고, 키는 179.8 이다.
